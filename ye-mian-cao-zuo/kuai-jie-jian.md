# 快捷键



| 描述               | 快捷键                |
| ---------------- | ------------------ |
| 保存               | Ctrl + S           |
| Run              | Ctrl + Enter       |
| DEBUG            | Ctrl + Alt + Enter |
| 代码提示             | Alt + /            |
| 单行注释             | Ctrl + /           |
| 多行注释             | Ctrl + Shift + /   |
| 编辑模式与postman模式切换 | Alt + w            |
|                  |                    |
